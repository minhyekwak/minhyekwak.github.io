1. Answering Machine with Emojies

This arduino project is an answering machine with emojies. Audience can ask any question to the artist(Ting), then Ting moves three switches between yes and no, straight forward and awkward, flirt and pissed. After she presses send button, she wears the mask, which is attached on other part of machine. Then the answer(emoji) is printed out of that machine. The algorythm is like this; Ting set 8 emojies to be printed, according to the choice of 3 options of switches.

2. This is Not a Piano

This project is similar to my arduino project in terms of using piano keyboard to sound like piano. But this keyboard used copper tape to be conductive. When the keyboard is pressed, the copper tape attached to the keyboard will reach to another copper tape attached to the bottom side. Then it will make a piano sound from the software of laptop. 7 notes were used to make sounds.

3. Grow your Virtual Tree

This beautiful machine contains a sense of nature and asian aesthetic. The machine made by wood, and 4 sides of small screen are included on the top side. Transparent acrylic panels divided 4 sides of each screen. In each screen, four tress are repeating to grow and shrink according to three functions of sticks(switches). If a user moves the first stick, the raindrops are falling in the screen. Second stick is for the sun, and if user input something in the hole on the center, the tree is growing.
