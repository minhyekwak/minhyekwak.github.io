For my first Javascript project, I made an website which contains fun and useful information of coffee. This website includes quotes about coffee, explanation of different kinds of coffee, recipes, famous coffeehouse in NYC, and other quotes from coffee essay. Each contents are generated when the page is loaded, so that users can see the different information and images in each time the page is refreshed. Besides, recipes, coffee shop info, and sentences of coffee essay are generated by clicking the buttons. JQuery, Tracery, and Markov were used.
