For my second Javascript project, I made an website which makes users bring a reminiscence of disney animation. Using p5.js, the website plays scenes and sounds of each animation when the user presses A to Z keys. For example, if a user presses A, the main scene of animation "Aladin" is played and "A Whole New World" song is coming out. With this playful functions, users can enjoy the memories of disney animations.

Visit "Into the Memories of Disney!"
