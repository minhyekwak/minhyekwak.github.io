For my third Javascript project, I made an website which facilitates real-time interaction between users remotely by using socket.io. Published in heroku, the website offers characters of Wes Anderson's films and allows users to color them. Users can pick a character and color they like, reset and erase functions are also provided. Users can collaborate to color the characters using different browsers or different devices at the same time. This website gives a feeling of connection and collaboration with fun activities.

Visit "Color Me" Website and have fun with your friend!
(for the best use, Chrome browser is recommended.)
