For my fourth Javascript project, I made an website uses Google Map API and OpenWeatherMap API to load content dynamically. This website contains the map and weather content of the cities I visited. Once the user picks one city among the list and inputs, then they can see the real-time weather (general, temperature, humidity, wind-speed, and cloud) by clicking the city in the map area. The information of weather including icons are called from the API, and the background images were made by myself and also called by the text.
